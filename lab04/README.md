### Базовый датчик случайных чисел

**Задание:**
- реализовать базовый датчик случайных чисел;
- вычислить выборочные среднее и дисперсию:
  - для реализованного датчика;
  - для встроенного генератора языка программирования;
- размер выборки — **100 000 значений**;
- сравнить результаты с теоретическими;
- сделать вывод.

### Параметры генератора:
* Алгоритм: мультипликативный конгруэнтный генератор
* Модуль ($M$): $2^{63}$
* Множитель ($\beta$): $2^{32} + 3$
* Размер выборки ($N$): 100 000 значений
* Начальное значение (seed): фиксированное ($2^{32} + 3$) для обеспечения воспроизводимости результатов

### Формулы расчета
Генерация последовательности:
1. $x_i^* = (\beta \cdot x_{i-1}^*) \mod M$
2. $x_i = \frac{x_i^*}{M}$

Статистические показатели:
1. Выборочное среднее: $\bar{x} = \frac{1}{n} \sum_{i=1}^{n} x_i$ (теория: 0.5)
2. Выборочная дисперсия: $S^2 = \frac{1}{n} \sum_{i=1}^{n} (x_i - \bar{x})^2$ (теория: $1/12 \approx 0.083333$)

### Результаты тестирования
Сравнение проводилось между теоретическими значениями, реализованным мультипликативным датчиком и встроенным генератором `System.Random`.

| Показатель | Теория | Мультипликативный | Встроенный |
| :--- | :---: | :---: | :---: |
| Выборочное среднее | 0.500000 | 0.500614 | 0.501212 |
| Выборочная дисперсия | 0.083333 | 0.083469 | 0.083680 |
| Погрешность среднего | 0.000000 | 0.000614 | 0.001212 |

### Выводы
1. Реализованный датчик показал высокую точность. Отклонение выборочного среднего от теоретического ($0.5$) составило всего 0.000614, что подтверждает свойство равномерности распределения на большой выборке.
2. Показатели реализованного мультипликативного датчика оказались сопоставимы по точности с `System.Random`. В данном эксперименте погрешность самописного датчика оказалась даже меньше, чем у встроенного.
